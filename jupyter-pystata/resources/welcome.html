Gen3 PyStata Workspace

About
This workspace is equipped to run Stata code in a Jupyter notebook environment using a shared Gen3 license. There are 5 active kernels in this workspace, each of which is authorized and running a Stata session. Please note, the Stata software and license used in this workspace are private, and belong to the Center for Translational Data Science at the University of Chicago. They are not to be copied or used outside of this workspace. Users with their own active Stata licenses are encouraged to use the bring-your-own-license Stata workspace.
Usage
Users can run Stata through any of the 5 existing interactive sessions. For detailed information on running Stata code from within Python, please see the [official pystata documentation](https://www.stata.com/python/pystata/notebook/Quick%20Start0.html);
